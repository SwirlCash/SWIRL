### Installation

Before running any command, make sure to install all dependencies:

```console
$ npm install
```

### Test

Run all tests via Truffle:

```console
$ truffle test
```

### Compile

Compile all contracts via Truffle:

```console
$ truffle compile
```

### Deploy

Deploy on local Truffle Develop blockchain:

```console
$ truffle develop
$ migrate
```
